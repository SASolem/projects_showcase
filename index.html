NeNorvalls Project Showcase

Contact-Us-Web-App

The "Contact Us Web App" is built using HTML, CSS, and JavaScript.

HTML provides the structure, CSS handles the styling, and JavaScript is used for client-side input validation and interactivity.

Onboarding Page Navigation

This project is a straightforward onboarding web application created using HTML, CSS, and JavaScript.

Dog Photo Scrapbook

This project is a dog photo scrapbook web page designed for users to browse through multiple pages of dog photos with the use of "Previous" and "Next" buttons.

The project primarily relies on HTML, CSS, and JavaScript to provide its functionality.

Timeline

This project, comprised of HTML and CSS, delivers an engaging and visually appealing timeline webpage with dynamic styling.

Ice Creams

The project is composed of HTML, CSS, and JavaScript files to create an interactive ice cream shop experience.

It enables users to view available ice cream flavors, manage their shopping cart, and see the total price of their selected items.

Long Light

The "Long Light" Landing Page is built using HTML for structure, CSS for styling, and JavaScript for interactive features such as the image slider. It utilizes these core web technologies for a visually appealing and user-friendly experience.

Image Slider

The "Image Slider Project" is a web project created with HTML, CSS, and JavaScript.

It features an interactive image slider with responsive design, automatic slideshows, navigation buttons, custom transition effects, and dynamic background animation

To-do-List

This project is a simple To-Do List application developed using HTML, CSS, and JavaScript.

Users can manage tasks seamlessly by adding, editing, and deleting them, with task data persistently stored in the browser's local storage.

Quiz App

This repository contains an interactive quiz implemented in HTML, CSS, and JavaScript. The quiz dynamically generates questions and allows users to submit their answers for evaluation.
